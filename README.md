# 🐢 小乌龟的小窝

一个具有科技感的个人主页，互动元素丰富，华丽但不繁杂。

## ✨ 特性

- 🌌 粒子背景动画 + 连线效果
- 🖱️ 鼠标跟随光晕
- 🃏 卡片 3D 倾斜效果
- ⌨️ 打字动画 + 数字滚动计数
- 📊 技能进度条动画
- 🎬 滚动触发动画
- 📱 响应式设计，适配移动端

## 🛠️ 技术栈

- HTML5
- CSS3（渐变、动画、玻璃态效果）
- JavaScript（Canvas 粒子系统、Intersection Observer）

## 🚀 使用方式

直接打开 `index.html` 即可预览，或启动本地服务器：

```bash
python3 -m http.server 8000
```

访问 http://localhost:8000

## 📁 项目结构

```
├── index.html    # 页面结构
├── style.css     # 样式文件
└── script.js     # 交互脚本
```
